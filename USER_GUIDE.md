## Login

1. Go to: **https://dashboard.vivadesignerboutique.org/login**
2. Enter your **Email/Username** and **Password**
3. Click **"Login"** button
4. You will be redirected to the Dashboard

**Note:** Only authorized users can access the dashboard. Contact your admin if you don't have credentials.

## Add Customer

### Steps to Add a New Customer:

1. **Navigate to Customers Module**
   - Click on **"Customers"** in the left sidebar

2. **Click "Add Customer" Button**
   - Look for the **"+ Add Customer"** button in the top right corner
   - Only admins and staff can add customers

3. **Fill in Customer Details**
   - **Customer Name:** Enter full name (required)
   - **Mobile Number:** Enter 10-digit phone number (required)
   - **Address:** Enter customer address (required)

4. **Submit**
   - Click **"Save"** button
   - A success message will appear
   - Customer is now added to the list

### View Customers:
- All customers are displayed in a table
- **Search:** Use the search bar to find customers by name or mobile number
- **Edit:** Click the pencil icon to edit customer details
- **Delete:** Click the trash icon to delete (admin only)
- **Measurements:** Click the ruler icon to add/view measurements
- **Print:** Click the print icon to print measurement sheet

## Add User

### Steps to Add a New User (Admin Only):

1. **Navigate to Users Module**
   - Click on **"Users"** in the left sidebar
   - **Note:** Only Admins can see and manage users

2. **Click "Add User" Button**
   - Look for the **"+ Add User"** button in the top right corner

3. **Fill in User Details**
   - **Username:** Enter username (minimum 6 characters, required)
   - **Password:** Enter password (minimum 6 characters, required)
   - **Role:** Select role from dropdown:
     - **Admin:** Full access to all features including user management
     - **Staff:** Can add/edit customers and measurements, but cannot manage users
     - **Tailor:** Read-only access to view measurements only

4. **Submit**
   - Click **"Save"** button
   - A success message will appear
   - User is now added to the system

### View/Delete Users:
- All users are displayed in a table
- **Search:** Use the search bar to find users by username
- **Delete:** Click the trash icon to delete a user (admin only)

## Add Measurement

### Steps to Add Customer Measurements:

1. **Navigate to Customers Module**
   - Click on **"Customers"** in the left sidebar

2. **Select a Customer**
   - Find the customer in the list
   - Click the **Ruler Icon** (üìè) to add measurements

3. **Measurement Modal Opens**
   - You will see two tabs:
     - **Blouse / Top**
     - **Lehenga / Pant**

4. **Fill Blouse/Top Measurements:**
   - Enter values for:
     - Blouse Length
     - Kurta Length
     - Upper Chest
     - Chest
     - Waist
     - Hip
     - Shoulder
     - Sleeve Length
     - Mori
     - By Shape
     - Armhole
     - Front Neck Depth
     - Back Neck Depth
     - Dart Point
     - Shoulder Seam
     - Neck Circumference
   
   **Validation Rules:**
   - Values must be between 0-100
   - Only numeric values allowed
   - Cannot enter letters or scientific notation

5. **Fill Lehenga/Pant Measurements:**
   - Enter values for:
     - Length
     - Waist
     - Hip
     - Thigh
     - Knee
     - Ankle
     - Drawstring

6. **Save Measurements**
   - Click **"Save"** button
   - Loading indicator will show while saving
   - Success message will appear

7. **Note:**
   - You can save partial measurements (e.g., only Blouse without Lehenga)
   - Empty fields are automatically ignored
   - Measurements are updated, not duplicated

## Troubleshooting

### I can't log in
- Verify your email/username is correct
- Check if your password is correct
- Contact your admin if account doesn't exist
- Clear browser cache and cookies and try again

### I can't see the Users menu
- Only Admins can see the Users menu
- Ask your admin to verify your role
- Re-login after role change

### Measurement values not saving
- Check that all values are between 0-100
- Ensure no letters or special characters are entered
- Check your internet connection
- Try refreshing the page

### Can't delete customer or user
- Only Admins can delete
- Contact your admin if you need something deleted
- Make sure you're not trying to delete yourself
